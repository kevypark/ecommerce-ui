## Building and running on localhost

First install dependencies:

```sh
npm install
```

To create a production build:

```sh
npm run prod:build
```

To create a development build:

```sh
npm run dev:build
```

## Running 

localhost:9000

To run in development mode:

```sh
npm run dev:start
```
